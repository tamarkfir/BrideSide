# BrideSide

מנחה AI שעוזרת לזוגות לסביות לבנות **סקראפבוק** — מחברת מעוצבת, עמוד-עמוד — שמתוכה נולד טקס החתונה שלהן. המנחה שואלת, מציעה שירים וברכות מותאמות, מייצרת איורים, ועורכת תמונות שהן מעלות.

## הרצה

1. התקנת תלויות (פעם אחת):

   ```
   npm install
   ```

2. הזנת מפתח API ב-[`.env.local`](.env.local):

   ```
   GEMINI_API_KEY=...
   ```

   מפתח מ-[Google AI Studio](https://aistudio.google.com/apikey). בלי מפתח האפליקציה עדיין עובדת מקצה לקצה — עם נוסחי גיבוי קבועים במקום תגובות המנחה החיות, ובלי איורים/עריכת תמונות.

3. הרצה:

   ```
   npm run dev
   ```

   ואז לפתוח את http://localhost:3000

## איך זה בנוי

הסקראפבוק הוא רשימת **עמודים** מסודרת ([`lib/sections.ts`](lib/sections.ts)) שמזינה רכיב עמוד גנרי אחד ([`components/notebook/ScrapbookPage.tsx`](components/notebook/ScrapbookPage.tsx)). כל עמוד יכול לכלול: הזמנה לתמונות, משימת כתיבה, רגע reveal, הצעות תוכן מהמנחה, ואיור — והכל נגזר מ-seeds, כך שהמנחה מאלתרת ולא מקריאה טקסט קבוע.

| מה | איפה |
|---|---|
| מבנה העמודים (ה"השראה" של המנחה) | [`lib/sections.ts`](lib/sections.ts) |
| כל ה-prompts + אישיות המנחה + נוסחי גיבוי | [`lib/prompts.ts`](lib/prompts.ts) |
| מודל הטקסט (`gemini-2.5-flash`) | [`app/api/chat/route.ts`](app/api/chat/route.ts) |
| מודל התמונות — איורים + עריכה (`gemini-2.5-flash-image`) | [`app/api/image/route.ts`](app/api/image/route.ts) |
| צבעים, פונטים, מרקם נייר וסגנון סקראפבוק | [`tailwind.config.ts`](tailwind.config.ts) + [`styles/globals.css`](styles/globals.css) |
| מבנה ה-state | [`lib/types.ts`](lib/types.ts) + [`lib/state.ts`](lib/state.ts) |

## עמודי הסקראפבוק

מי אנחנו · הסיפור שלנו · מה שבינינו · שורשים ומסורת · מילים ושירים · הטקס שלנו (ההחלטות) · כוונה.

הקשר קודם לרקע הדתי; הברכות המסורתיות מותאמות מגדרית לשתי כלות וניתנות למשיכה לכיוון אישי יותר; עמוד ההחלטות מרכז את ההכרעות לטקס (כניסה, מסורות בפנים/בחוץ, איזון דתי-אישי, מי עורכת, מי על החופה). הסיום מייצר ספרון להדפסה / שמירה כ-PDF.

## הערות

- **תמונות** — מעוצבות בדפדפן (מסגרות, סרט הדבקה, סיבוב, כיתוב) ונשארות על המכשיר. כפתור "AI restyle" לכל תמונה (כולל גזירת רקע למדבקה) שולח את אותה תמונה ל-Gemini רק בלחיצה.
- **בלי DB** — ה-state בזיכרון בלבד בשלב זה.
